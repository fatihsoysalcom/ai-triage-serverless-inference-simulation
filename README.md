# AI Triage Serverless Inference Simulation

This example demonstrates the core logic of an AI-powered customer support triage system. It simulates how an AI model would classify incoming support messages into categories and assign priorities, suggesting appropriate actions. The code conceptually highlights how such a classification function would be executed as a serverless inference, enabling cost-effective and scalable AI solutions by paying only for actual usage.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
3. Enter customer support messages at the prompt to see the simulated AI triage results.
